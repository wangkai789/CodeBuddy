# 病案业务报表汇总 V2 需求文档

## 概述
从医疗业务报表Excel文件中按科室搜索数据，进行汇总处理后输出Excel文件并发送到钉钉。

## 数据源
- 路径: `D:\文档\正骨医院\20260401病案业务报表查询\data`
- 文件: `2025医疗业务报表.xlsx`
- 读取方式: header在第4行

## 功能需求

### 1. 命令行参数
- 输入: 科室名称关键词（支持模糊匹配）
- 用法: `python 病案业务报表汇总.py 风湿病`

### 2. 数据字段
输出Excel包含以下字段：
| 字段 | 说明 |
|------|------|
| 年度 | 从文件名提取（如2025） |
| 科室 | 科室名称 |
| 门急诊人数 | 门急诊人数 |
| 门急诊人数(不含零费用) | 门急诊人数(不含零费用) |
| 出院人次 | 出院人数 |

### 3. 数据处理
- **保留原始数据**: 门急诊人数和门急诊人数(不含零费用)两列分开显示，数据不进行处理

### 4. 科室名称规范化
- 去掉科室字段中的前缀"A"
- 将"风湿病三科"修改为"风湿病三科(骨质疏松科)"
- 处理全角括号情况，统一使用中文括号`（）`

### 5. 数据汇总
- 按年度+科室分组汇总
- 按科室数字顺序排序（如：风湿病一科 → 风湿病二科 → 风湿病三科）
- 添加合计行

### 6. Excel格式化
- 表头加粗、居中
- 所有单元格添加边框
- 设置列宽：
  - 年度: 10
  - 科室: 20
  - 门急诊人数: 16
  - 门急诊人数(不含零费用): 24
  - 出院人次: 12
- 设置行高：表头20，数据行18

### 7. 钉钉通知
- 将结果以Markdown表格形式发送到钉钉机器人

## 输出文件
