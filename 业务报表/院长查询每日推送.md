# 院长查询每日推送

## 概述
从InterSystems IRIS数据库查询当日门诊、急诊、入院人次数据，发送到钉钉。

## 数据库连接
- 服务器: `172.20.9.3`
- 端口: `51773`
- 用户名: `dhsuper`
- 密码: `o8kfmm#2P`
- 数据库名称: `dhc-app`

## 查询内容
| 指标 | 说明 |
|------|------|
| 门诊人次 | 当日门诊就诊人数 |
| 急诊人次 | 当日急诊就诊人数 |
| 入院人次 | 当日入院人数 |

## 执行方式
```bash
python d:/codebuddy/院长查询每日推送.py
```

## 示例输出
| 指标 | 数量 |
|------|------|
| 门诊人次 | 1234 |
| 急诊人次 | 456 |
| 入院人次 | 78 |

## 定时任务
