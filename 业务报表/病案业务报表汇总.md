
# 病案业务报表科室汇总程序需求文档

## 功能概述
从病案业务报表目录中查询Excel文件中按科室搜索记录，按年度汇总后输出Excel文件，并支持发送到钉钉。

## 数据源
- 目录：`D:\文档\正骨医院\20260401病案业务报表查询\data`
- 文件列表（1个Excel文件）：
  - 2025医疗业务报表.xlsx    
  
## 需求清单

### 1. 搜索医生记录
- 从data目录下所有Excel文件中搜索指定科室的记录
- 支持通过命令行参数指定科室名称：`python  d:/codebuddy/病案业务报表汇总.py 科室名称`
- 也支持运行时输入科室名称

### 2. 按年度汇总
- 将搜索到的记录按年度汇总
- 汇总字段：科室、门急诊人数、门急诊人数(不含零费用)、出院人次

### 3. 输出Excel文件
- 输出文件路径：`D:\文档\正骨医院\20260401病案业务报表查询\{科室名称}.xlsx`
- 输出格式：
  - 表头加粗、居中对齐
  - 带有表格边框
  - 设置合适的列宽和行高

### 4. 发送到钉钉
- 支持将汇总结果发送到钉钉机器人
- 钉钉Webhook：`https://oapi.dingtalk.com/robot/send?access_token=d68e1626c9e5689513089f2d67a9b698067442884f342c733b82e953abb71df7`
- 发送格式：Markdown表格形式

## 运行方式

```bash
# 方式1：命令行指定科室名称

python d:/codebuddy/病案业务报表汇总.py 风湿病


# 方式2：运行时输入科室名称
python d:/codebuddy/病案业务报表汇总.py
```
