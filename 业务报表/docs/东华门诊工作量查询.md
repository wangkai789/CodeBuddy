# 东华工作量查询程序需求文档

## 功能概述

从医院职称数据Excel文件中按医生姓名搜索记录，按期间（年度/月度）汇总后输出Excel文件，并支持发送到钉钉。

  - 2020医生组出入院统计报表.xlsx
  - 2021医生组出入院统计报表.xlsx
  - 2022医生组出入院统计报表.xlsx
  - 2023医生组出入院统计报表.xlsx

  - 202601-03医生组出入院统计报表.xlsx

## 需求清单

### 1. 搜索医生记录
- 从data目录下所有Excel文件中搜索指定医生姓名的记录
- 支持通过命令行参数指定医生姓名：`python 程序名.py 医生姓名`
- 也支持运行时输入医生姓名

### 2. 按期间汇总
- 将搜索到的记录按期间汇总（年度或年月）
- **特殊处理**：2026年的数据显示为"2026-01~03"格式（精确到月份），其他年份按年度显示
- 汇总字段：门诊人次、出院人次

### 3. 门诊人次取值规则
- 当"门诊人次(不含零费用)"列有值且大于0时 → 使用该值
- 否则 → 使用"门诊人次"的值
- 逻辑代码：
```python
outpatient_no_zero = pd.to_numeric(matched['门诊人次(不含零费用)'], errors='coerce')
outpatient = pd.to_numeric(matched['门诊人次'], errors='coerce')

if outpatient_no_zero.notna().any() and outpatient_no_zero.sum() > 0:
    yearly_data[period]['门诊人次'] += outpatient_no_zero.sum()
else:
    yearly_data[period]['门诊人次'] += outpatient.sum()
```

### 4. 输出Excel文件
- 输出文件路径：`D:\文档\正骨医院\20260226职称数据\{医生姓名}.xlsx`
- 使用临时文件方式保存（先保存为*_temp.xlsx，再重命名），避免文件占用问题
- 输出格式：
  - 表头加粗、居中对齐
  - 带有表格边框（细边框）
  - 列宽设置：A列(期间)14、B列(门诊人次)22、C列(出院人次)12
  - 行高设置：表头20px、数据行18px

### 5. 发送到钉钉
- 支持将汇总结果发送到钉钉机器人
- 钉钉Webhook：`https://oapi.dingtalk.com/robot/send?access_token=d68e1626c9e5689513089f2d67a9b698067442884f342c733b82e953abb71df7`
- 发送格式：文本消息形式，包含每个期间的门诊人次和出院人次

### 6. 数据读取适配
- 不同年份的Excel文件表头位置不同：
  - **2019~2022年**：表头在第2行（header_row=2）
  - **2023年及以后**：表头在第4行（header_row=4）
- 自动处理列名映射：
  - '入院人数' → '入院人次'
  - '出院人数' → '出院人次'
  - '出院患者占床日' → '出院患者占用床日数'
