# 东华住院量查询

## 概述
从SQL Server数据库查询工作量数据（医师、护士、麻醉师）

## 数据库连接
- 服务器: `192.168.101.19`
- 端口: `1433`
- 用户名: `dhuser`
- 密码: `dhcc`
- 数据库名称: `dhcc`

## 查询存储过程

### 1. 正高职称医师
```sql
EXEC dhc..Search_Doctor_SP_Workload '王治国'
```

### 2. 副高职称医师
```sql
EXEC dhcc..Search_Doctor_AP_Workload '王帅'
```

### 3. 护士工作量
```sql
EXEC dhcc..Search_Nurse_Workload '王淑梅'
```

### 4. 麻醉工作量
```sql
EXEC dhcc..Search_Anes_Workload '赵丽艳'
```

## 程序文件

### 命令行版本
- 路径: `src/东华住院量查询.py`
- 运行: `python src/东华住院量查询.py`

### Web版本
- 主程序: `web_app_inpatient.py`
- 端口: `5003`
- 模板: `templates/inpatient_query.html`
- 运行: `python web_app_inpatient.py`

## 功能特性

### Web界面功能
1. 输入姓名查询
2. 选择查询类型（正高医师/副高医师/护士/麻醉师）
3. 修改数据库配置
4. 查询历史记录
5. 打开输出目录

### 输出
- Excel文件格式
- 自动保存到配置目录
- 文件名格式: `{姓名}_{类型}_{日期}.xlsx`
