## 用户需求

读取 `D:\文档\正骨医院\20260226职称数据\data` 目录下所有8个Excel文件，搜索包含"沙莎"的记录，并将结果写入 `D:\文档\正骨医院\20260226职称数据\沙莎.xlsx`。

## 数据源文件

目录下共8个 `.xlsx` 文件（2019-2025年医生组出入院统计报表 + 202601-02月报表），结构统一：第5行为表头（科室、医生组、医生、门诊人次、门诊人次(不含零费用)、入院人次、出院人次、实际占用床日数、出院患者占用床日数），第6行起为数据。

## 输出要求

- 输出文件：`D:\文档\正骨医院\20260226职称数据\沙莎.xlsx`
- 结果包含：序号、来源文件、科室、医生组、医生、门诊人次、门诊人次(不含零费用)、入院人次、出院人次、实际占用床日数、出院患者占用床日数
- 之前执行结果为17条记录（覆盖2019-2025年所有年份）

## 技术方案

- 使用 Python + pandas 读取所有 xlsx 文件
- 使用 [skill:xlsx] 技能规范创建输出 Excel 文件（openpyxl 格式化）
- 脚本逻辑：遍历所有文件 -> 动态获取sheet名称 -> 跳过前4行 -> 逐行搜索"沙莎" -> 收集结果 -> 写入 xlsx

## 实现说明

- 之前已安装 pandas 和 openpyxl，无需重复安装
- 需要动态获取每个文件的sheet名称（之前发现部分文件sheet名称不是"sheet1"）
- 从Row 4获取表头行，Row 5起为数据行
- 表头列：科室、医生组、医生、门诊人次、门诊人次(不含零费用)、入院人次、出院人次、实际占用床日数、出院患者占用床日数
